# Contributing

Thank you for contributing. Before opening a PR, please read the rules:

1. **Purpose**: Submit only material for defensive, educational, or research purposes.

2. **Communication**: Open an issue describing the intent before PRs that add sensitive material.

3. **Reviews**: PRs will be reviewed for operational and legal risk; content that facilitates attacks will be rejected.

4. **Tests & Docs**: Include documentation and, when applicable, tests or inert examples.
